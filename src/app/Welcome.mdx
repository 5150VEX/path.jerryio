## Introduction

PATH.JERRYIO is a multi-purpose path editor/planner which can be used to design routes for one-minute driver skill and generate path files.
More information about the editor can be found on the [GitHub Page](https://github.com/Jerrylum/path.jerryio).


## Getting Started

Start using the editor by left clicking on the field to create a new path with two end points.

<br/><br/><br/>

Add a curve or a linear spline simply by left clicking or right clicking on the field.

Undo/redo any modification with Ctrl+Z/Ctrl+Y or select Undo/Redo in the menu

Add more end points and drag them to shape the path.

Delete a control by right clicking.

To move multiple controls at the same time, hold shift and left click the controls or select the area of the field with/without holding shift key.

Press shift to start dragging multiple controls, shift key can be released once it is start dragging.

While you are dragging control(s), press shift to start

Only end points have heading attributes. Modify ...

Edit the exact coordinate and/or heading in the input fields.

Change the default unit of length in the configuration panel.

Convert spline(part of the path) into line or spline by left click the path.

Add a control in the middle of the path by right click the path.

Save the path file with Ctrl+S or select "Save" in the menu
